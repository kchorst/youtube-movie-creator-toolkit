# Local LLM Voice Guide

The metadata workflow can use an optional Local LLM for draft titles, descriptions, tags, and regeneration. It is provider-neutral.

Use Settings to configure:

- LLM mode: manual only or Local LLM
- Local LLM endpoint
- Model name
- Voice/style override
- Examples override

Use **Find** to auto-detect llama-server, LM Studio, Ollama, or a saved custom endpoint.

Manual metadata editing remains available when no Local LLM is running.
